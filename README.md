# ML Test Model

An educational machine learning project demonstrating a complete
workflow for a simple ML model: from loading and analyzing data
to training a Linear Regression model and evaluating its performance.
## Technologies

- Python 3.12
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Project Goal

The goal of this project is to build a simple machine learning
pipeline that predicts a person's height based on age using Linear Regression

The project demonstrates the complete workflow:

- loading data;
- preprocessing data;
- splitting the dataset into training and testing sets;
- training a Linear Regression model;
- evaluating model performance.


## Dataset

The project uses a small CSV dataset containing information 
about people's age and height.

Features:

- Age (input feature)

Target:

- Height (target variable)


## Results

The Linear Regression model was successfully trained and evaluated
on the dataset.

The project demonstrates the complete machine learning workflow,
including data preprocessing, model training, prediction, 
and performance evaluation using regression metrics.

## How to Run

1. Clone the repository.
2. Install the required libraries.
3. Run the Python scripts from the project directory.
